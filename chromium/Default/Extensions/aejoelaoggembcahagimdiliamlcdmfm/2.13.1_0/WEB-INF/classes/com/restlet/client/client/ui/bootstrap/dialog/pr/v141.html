January 12, 2017 - DHC 1.4.1

Enhance your headers

DHC now supports setting assertions on headers that are present multiple times in the response.

For example, let's say that header "Cache-Control" is received as follow:

            Cache-Control: no-cache
            Cache-Control: no-store

You can check that the header "Cache-Control" contains the value "no-cache" or "no-cache,no-store". As the order matters, it does not contain the value "no-store,no-cache".

Various bug fixes and improvements have also been made across the application.
