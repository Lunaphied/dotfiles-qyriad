January 27, 2017 - Restlet Client 1.4.2

DHC will become Restlet Client soon!

Dear user,
First of all, thank you for using DHC! We developed DHC for many years with love to bring you the best tool for API testing and orchestration.

A year ago DHC started its journey to become part of the Restlet’s API-First Platform. In the next few weeks DHC will become Restlet Client and get a new logo, but the tool itself won’t change, just continue to get better and listen to your feedback.

The goal of the Restlet’s API-First Platform is to help you with APIs from design to documentation and testing, all the way through to deployment. You can use your Restlet account to try out Restlet Studio and Restlet Cloud (fka APISpark) modules.
