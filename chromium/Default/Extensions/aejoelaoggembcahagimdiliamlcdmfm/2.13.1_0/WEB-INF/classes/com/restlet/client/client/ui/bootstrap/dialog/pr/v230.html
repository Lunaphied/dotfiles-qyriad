May 2, 2017 - Restlet Client 2.3.0

Building API Scenarios Has Never Been Easier

Many of your recent feedback was focused on how to improve the Scenario creation experience. In this release we’ve implemented a set of features to help you build Scenarios from different sources:

  • With the new Extract to scenario feature, available from projects or services contextual menus, building a scenario from existing requests has never been easier.
  • You can now add a bunch of requests in one go to complete an existing Scenario.
  • Save from History has been improved, we hope you’ll find it helpful.
  • Last but not least, we’ve enhanced the HAR import to help you create Scenarios to test web app backends.

Please read the blog post to get more details.

We hope these improvements will save you time. Thank you for the feedback and keep it coming!
